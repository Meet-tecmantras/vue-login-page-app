# Vue Login Page App

This repository houses a focused Vue 3 login experience built with Vite. It demonstrates responsive layout, inline validation, and a mock submit handler that logs payload data for integration testing.

## Project structure

- `frontend/` – Vite-powered Vue application
  - `index.html` – entry HTML that mounts the Vue application
  - `src/components/LoginForm.vue` – the reusable login form
  - `src/styles/global.css` – shared CSS tokens and background gradient
  - `package.json` – scripts and dependencies

## Getting started

```bash
cd frontend
npm install
npm run dev
```

`npm run dev` launches the Vite dev server (default port 5173). Use `npm run build` to produce a production-ready bundle and `npm run preview` to inspect the output.
